How it works

Connect, build, arm.

01

Connect

Log in to Kite or Breeze the way you already do — OAuth in your browser, TOTP and all. ThetaPrime stores the resulting token, not your password, and a refresh each morning is one click.

02

Build

Click blocks together in the Composer — structure, strikes, exits, sizing. The spec preview validates every parameter as you go, and a payoff diagram with one-click AI review helps you sanity-check the shape before you save.

03

Arm

Every strategy starts in Paper. Flip it to Real, then Arm it — two separate, deliberate clicks. Restart the app and it's unarmed again, every time, no exceptions.

Safety rails

The discipline built in, not remembered.

This is the part a solo Python script usually skips. It's the part that matters most.

Arming

Real ≠ live

A strategy needs to be set to Real and Armed before an order goes out. Arming resets to off on every restart, so an unattended reboot can never quietly start trading.

Loss limit

Daily stop, whole account

Set a hard stop in points, rupees, or both. The instant combined real-mode P&L crosses it, every Real strategy disarms itself and Real mode locks for the rest of the day.

Entry guard

No double entry

A strategy already holding a position can't open a second one. No accidental doubling from a re-triggered signal or a stray manual click.

Sizing

Hard lot cap

A ceiling on lot size per strategy, enforced before the order is placed. A misconfigured strategy gets silently capped, never silently oversized.

Reconciliation

Orphan detection

Every five minutes during market hours, and at startup, ThetaPrime diffs your broker's live positions against its own trade state. A crash or a manual square-off shows up as a red badge, not a surprise next week.

Auth

Expired login, no live orders

No fresh broker token, no live trading — full stop. Paper trading and backtests keep running regardless, so a stale login degrades gracefully instead of silently.

Login check

Broker API validated on login

Every Kite or Breeze token is checked against the exact read-only endpoints strategies depend on, the moment you log in. A broken or changed broker API fails at Auth, not mid-strategy at 9:20am.

Execution

Fill confirmation, not assumption

A real order isn't done until the broker confirms the fill. An unconfirmed or partial entry rolls back — closing whatever filled, cancelling what's still open — instead of leaving a silent, unhedged position.

FAQ

Before you ask.

Is ThetaPrime a broker or an investment adviser?
No. It's a self-hosted automation tool that places orders through your own Zerodha or ICICI Direct account. You choose the strategies and carry SEBI compliance responsibility; Kite Connect's API subscription is billed to you by Zerodha, separately from ThetaPrime.
Does my computer need to stay on during the trading day?
Yes. ThetaPrime runs as a Windows service on your own PC, not in any cloud — it can only enter, monitor, and exit positions while that machine is powered on and the service is running. A laptop that sleeps or shuts down stops managing your open trades until it's back.
What if my internet connection drops?
ThetaPrime needs a live connection to your broker for as long as a position is open — entries, monitoring, and exits all depend on it. Brief blips are retried automatically, but a sustained outage means it can't act on your positions until connectivity returns, the same as if you were watching a terminal yourself. A stable, always-on connection is part of the setup, not optional.
Do I need a static IP address?
Yes. SEBI's API trading rules require a fixed IP address for order placement, which you register with your broker alongside your API credentials. Most ISPs offer a static IP as a paid add-on to a home broadband plan — request one before setting up your Kite or Breeze API app.
Can I run one license on two computers?
No — one license, one machine. Buy a second license for a second machine.
I reinstalled Windows. Do I need a new key?
Only if your network adapter changed. If the machine fingerprint on the Activation page still matches, your existing key works as-is.
What happens to my data if my license lapses?
Nothing is deleted. Strategies, trade history, and configuration all stay on disk exactly as they were. Renewing restores full dashboard access immediately — and any strategy the scheduler was already running kept running the whole time.
Do I need to know how to code?
No. Every strategy — a simple strangle, an iron condor, a calendar spread — is built entirely in the Composer by clicking blocks together. There's no code path to write or maintain.
